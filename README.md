# 📊 Power BI Sales Analytics Dashboard

Interactive Sales Dashboard built in Power BI with a modern Glassmorphism UI.

## 🖼️ Dashboard Preview
![Dashboard Preview](Screenshot%202026-04-05%20145122.png)

## 🛠️ Features
- **Interactive Slicers:** Filter by Region and Category.
- **Top 10 Analysis:** Identify Top 10 States and Sub-Categories.
- **Profitability Insights:** Visualize profit margins across different segments.
- **Modern UI:** Glassmorphism theme with outer shadows.
